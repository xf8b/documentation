# Eval Command

The `eval` command allows you to execute/evaluate code.

**This command is only available to bot administrators.**

---

## Usage

- NOTE: Required Parameters = <> and Optional Parameters = []

Usage: `{prefix}eval <code to evaluate>`

---

## Aliases

`evaluate`
