# Skip Command
## Info
The `skip` command skips the current song playing.

**Anyone can use this command.**

---

## Usage

* NOTE: Required Parameters = <> and Optional Parameters = []
Usage: `{prefix}skip [numberOfSongs (defaults to 1)]`
